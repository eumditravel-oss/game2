# 🍀 SUPER LUNCH GACHA

레트로 아케이드 감성의 점심 메뉴 가챠 + 내기 룰렛 게임

## 기능

- **🍚 카테고리 뽑기**: 카테고리(한식/중식/일식/양식/분식/면/고기/매운맛 등)를 고르고 GRAB! 버튼으로 캡슐 뽑기
- **✏️ 메뉴 뽑기**: 직접 추가한 메뉴만 넣고 돌리는 전용 가챠 (메뉴 추가/삭제, 브라우저에 자동 저장)
- **🎯 내기 룰렛**: 참가자 이름을 넣고 스핀 — 핀에 걸리는 긴박한 연출과 함께 "당첨자가 쏜다" 또는 "당첨자만 공짜" 모드
